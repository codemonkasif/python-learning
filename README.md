## 🚀 Project Overview

The goal of this repository is to create a structured path for learning Python. It covers everything from the "Hello World" basics to complex Data Structures and Object-Oriented Programming (OOP).

### 📁 Directory Roadmap

| Folder | Content |
| :--- | :--- |
| `01_Basics/` | Variables, Operators, and Input/Output. |
| `02_Logic/` | If-Else statements and Loops (For/While). |
| `03_Data_Structures/` | Lists, Tuples, Sets, and Dictionaries. |
| `04_Functions/` | Modular code, Arguments, and Lambda functions. |
| `05_OOP/` | Classes, Objects, Inheritance, and Decorators. |
| `06_Files_IO/` | Reading and writing Text/CSV/JSON files. |
| `07_Projects/` | Small real-world applications (Calculators, Scrapers, etc.). |
